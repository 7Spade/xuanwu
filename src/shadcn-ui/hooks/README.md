# src/shadcn-ui/hooks

shadcn/ui 配套 React Hooks。由 shadcn/ui CLI 生成並維護於此目錄。

## Files

| File | Exports |
|------|---------|
| `use-mobile.tsx` | `useIsMobile()` — 偵測行動裝置斷點（< 768px） |
| `use-toast.ts` | `useToast()`, `toast()` — Toast 通知狀態管理 |

## Usage

```ts
import { useIsMobile } from "@/shadcn-ui/hooks/use-mobile"
import { useToast, toast } from "@/shadcn-ui/hooks/use-toast"
```
